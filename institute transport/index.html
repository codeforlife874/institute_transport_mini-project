Algorithms Used
⚡
Dijkstra's Algorithm
Shortest Weighted Path
Uses a min-heap priority queue to greedily explore the node with smallest known distance. Guarantees optimal result on non-negative weighted graphs.
Time: O(E log V) Space: O(V)
🔍
BFS
Fewest Stops Path
Breadth-First Search explores all nodes at the current depth before moving deeper. Finds the path with the minimum number of intermediate stops.
Time: O(V + E) Space: O(V)
🎯
A* Search
Heuristic Path
Extends Dijkstra with a heuristic function h(n) to guide search towards the goal. With GPS coordinates (future), uses Euclidean distance as heuristic.
Time: O(E log V) Space: O(V)
